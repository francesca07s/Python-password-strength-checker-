# Password Strength Checker

This is a simple Python program I developed while learning how to program.

This program prompts the user to enter a password. Then it checks the strength of the password based on the length of the password.

A weak password is a password with a small length. A medium or strong password is a password with a longer length.

I developed this program to help me learn how to deal with user input, conditions, and logic in Python.

## What the Program Does

- Prompts the user to enter a password.
- Checks the strength of the password.
- Determines whether the password is weak, medium, or strong.
- Prints the result.

## Technologies Used

- Python

## Example

Password:
mypassword

Result:
Password strength: medium
